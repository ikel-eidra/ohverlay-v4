## Our Solution: Ohverlay

### Core Concept

Ohverlay places a lifelike AI Betta fish (named "Uno") on users' desktop monitors. The fish:

1. **Swims naturally** across multi-monitor setups using procedural animation
2. **Observes work patterns** (respectfully, locally, privately)
3. **Delivers wellness reminders** via gentle floating bubbles
4. **Respects focus time** through "Sanctuary Mode"
